# Claire: Your personalized trend analyzer.

## Description
Claire is a tool that utilizes machine learning and bayesian sentiment analysis to predict consumer reactions to trendy topics.

Claire's favorite inputs:
* Trump
* iPhone 7
* Finding Dory
